# ALZ OpenClaw Landing Page Specification

## Project Context
Landing page for ALZ's OpenClaw Fast Track service - a hosted AI coding tool solution targeting developers who want quick access without deployment hassles.

## Design System

### Color Palette (NO BLUE/PURPLE)
- **Primary Background**: #0a0a0a (Deep Black)
- **Secondary Background**: #141414 (Dark Grey)
- **Card Background**: #1a1a1a (Soft Black)
- **Accent Primary**: #10b981 (Emerald Green - Tech feel)
- **Accent Secondary**: #f59e0b (Amber - Warmth, urgency)
- **Text Primary**: #ffffff
- **Text Secondary**: #a1a1aa (Zinc)
- **Border**: #27272a

### Typography
- **Headlines**: Inter/System UI, Bold, 4xl-7xl
- **Body**: Inter/System UI, Regular, base-lg
- **Pricing**: Monospace for numbers

### Layout
- Full-width sections with max-w-6xl content
- Generous spacing (py-20 to py-32)
- Card-based pricing display

## Hero Section (Video-Led)
- Full-height cinematic video background
- Abstract tech/code visualization
- Massive typography overlay
- CTA button: "添加微信咨询" with WeChat ID

## Sections Structure
1. **Hero**: Main headline + subtitle + CTA
2. **Features**: 5 core selling points with icons
3. **Pricing**: 3-tier card layout (体验版/实用版/长租版)
4. **Comparison**: Table comparing vs other solutions
5. **Footer**: Contact info + final CTA

## Motion
- Subtle fade-in animations on scroll
- Hover effects on cards and buttons
- Video looping seamlessly

## Tech Stack
- Single HTML file with embedded CSS
- Vanilla JS for interactions
- No external dependencies except fonts
